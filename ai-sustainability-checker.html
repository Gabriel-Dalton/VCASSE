How this becomes a real product

1) Website widget

Embeddable component for schools, nonprofits, and media organizations to visualize AI footprint by activity type.

2) Chrome extension MVP

Popup-based tracker for prompt count and estimated water/energy/CO₂ impacts with local storage and CSV export potential.

3) Provider data API

Replace generic coefficients with platform-specific disclosures once providers publish operational sustainability metrics.

Chrome extension files included in this repo

See extension/ai-sustainability-checker/ for a ready-to-load Chrome extension MVP popup that mirrors this estimator and stores totals in browser storage.
